| Boundary | Description |
|----------|-------------|
| resolved CLI config -> narration request | User-authored config becomes provider request input |
| environment -> OpenAI adapter | Secret material enters the process only through environment variables |
| OpenAI adapter -> external speech API | Untrusted network responses are converted into local provider results |
