Implement against the Phase 2 SDK contract:
```typescript
type SmokeRuntime = DemoHunterRunContext & DemoHunterLifecycleContext;

export function createSmokeTourRuntime(args: {
  page: Page;
  outputDir: string;
  onEvent?: (event: SmokeTourRuntimeEvent) => void;
}): SmokeRuntime;
```

Helper behavior stays thin:
- `chapter(title)` records a marker and updates current chapter; no block DSL.
- `step(title, fn)` executes `fn()` immediately and returns its value.
- `narrate(text, opts?)` records a narration event only; no TTS work in Phase 2.
- `waitForStable`, `highlight`, `snapshot`, and `assertVisible` use Playwright-native primitives only.
